
# Calculator (Python)

A basic console-based calculator built in Python.  
It can perform addition, subtraction, multiplication, division, modulo, and power operations.

## How to Run
1. Install Python 3 on your system.
2. Clone this repository:
   ```bash
   git clone https://github.com/<your-username>/Calculator-Python.git
   cd Calculator-Python


RUN THE PROGRAM :
python calculator.py


EXAMPLE:
=== Calculator ===
1. Add
2. Subtract
3. Multiply
4. Divide
5. Modulo
6. Power
7. Exit
Choose an option (1-7): 1
Enter first number: 5
Enter second number: 3
Result: 8.0
